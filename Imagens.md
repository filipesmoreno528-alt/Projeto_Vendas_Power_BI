# Imagens dos Gráficos

Esta pasta contém as capturas de tela do dashboard desenvolvido no Power BI.

## Conteúdo

- Relatório de Clientes

As imagens servem para apresentar a interface do dashboard sem a necessidade de abrir o arquivo `.pbix`, facilitando a visualização do projeto por recrutadores e desenvolvedores.
